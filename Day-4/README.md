## Script 

Script can be multiple types:
- AWS CLI
    - Automate the creating of virtual machine or EC2 instance through AWS CLI
    - CLI -> Cloud Line Interface 
- AWS API
    - API -> Application Programming Interface
    - Create the RESTAPI we can deal with the AWS API some module will directly allow to make a call with API. For eg: (BOTO 3)
- AWS CFT
    - AWS support, templeting language will return n number of VM which is required.
    - CFT -> cloud Formation Templete
- AWS CDK
    - CDK -> Cloud Devlopment Kit
    - CDK lets you write AWS infrastructure using programming languages like:TypeScript,Python,Java,C#.
- Terraform
    - Terraform is a cloud-agnostic Infrastructure as Code tool.  
Meaning:
      - It can work with AWS, Azure, GCP, Kubernetes, VMware, Oracle Cloud, etc.
