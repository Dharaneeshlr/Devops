# What is Devops?

```
Devops is the process of improving the applicarion(script) delivery by ensuring there is a proper Automation, with code Quality that is maintained, ensuring continuous Monitoring(observability) and continous Testing in place.
````

# Why Devops?
```
DevOps helps deliver software faster, with fewer errors, more automation, better teamwork, and more reliable systems.

##Traditional model:
Developers write code
Operations deploy and maintain
Results in slow releases, miscommunication, and many bugs

##DevOps model:
Developers and operations teams work together
Use automation  
Release faster and more reliably
```
